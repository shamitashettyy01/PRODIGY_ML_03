# Cat vs Dog Image Classification using SVM

## 📌 Overview
This project is a part of my Data Science Internship at Prodigy InfoTech.  
The goal of this project is to build an image classification model that can distinguish between cats and dogs using Support Vector Machine (SVM).

---

## 🎯 Objective
To develop a machine learning model capable of classifying cat and dog images.

---

## 📂 Dataset
The dataset contains images of:
- Cats
- Dogs

---

## 🛠️ Technologies Used
- Python
- NumPy
- PIL
- Scikit-learn
- Jupyter Notebook

---

## ⚙️ Steps Performed
1. Imported required libraries  
2. Loaded image dataset  
3. Resized and preprocessed images  
4. Converted images into arrays  
5. Trained the SVM classification model  
6. Predicted image categories  
7. Evaluated model accuracy  

---

## 📊 Model Used
- Support Vector Machine (SVM)

---

## 📈 Outcome
Successfully developed an image classification model capable of identifying cats and dogs.
